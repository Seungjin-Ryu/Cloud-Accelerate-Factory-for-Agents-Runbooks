# Cloud Accelerate Factory (CAF) for Agents

> Microsoft's structured delivery framework designed to accelerate production-ready AI Agent deployments — from committed use case to live in production.

---

## 🎯 Purpose

The **Cloud Accelerate Factory (CAF) for Agents** is Microsoft's **single front door** for customers with committed, production-ready agent delivery scenarios.

CAF provides **zero-cost, hands-on engineering support** to accelerate the building and deployment of high-value agent use cases using **Copilot Studio** and/or **Microsoft AI Foundry**. Through structured 6–8 week sprint engagements, CAF leverages proven runbooks, reference architectures, and repeatable delivery patterns to bring agents from development to production — with confidence in security, governance, and scalability.

> 💡 If your customer does **not** yet have a committed use case and needs pre-sales development or trials, consider Copilot Acceleration and Trials (CAT) instead.

---

## 💡 Core Value

CAF delivers measurable value to customers and Microsoft field teams alike:

- **Zero-cost engineering support** — Microsoft-invested, hands-on-keyboard delivery at no incremental cost to the customer
- **Faster time to production** — Structured sprints with proven patterns reduce deployment timelines significantly
- **Deployment clarity** — Pre-defined, repeatable runbooks ensure consistent and reliable delivery outcomes
- **Security & governance by design** — Every agent is built to meet enterprise standards via the Agent Landing Zone framework
- **Expert access** — Customers gain direct access to Microsoft CSAs, Cloud Factory Technical Leads, and vetted delivery partners
- **Smart routing** — If a scenario falls outside CAF scope, customers are seamlessly routed to the most appropriate program (CSU, ISD, FDE, PG, or Partner)

## 🏢 Target Customers

CAF is available for **managed enterprise customers** (excluding SMB) who meet the following qualification criteria:

| Criteria | Details |
|---|---|
| **Committed use case** | Customer must have a confirmed, production-ready agent deployment scenario |
| **Sales Stage** | Opportunity at Sales Stage 2 or later in MSX |
| **Eligible workloads** | Must have an active milestone in one of the supported workloads (see below) |
| **Deployment readiness** | Customer must be ready to deploy — not in early discovery or pre-sales |
| **ACV threshold** | \\$50K ACV or projected \\$50K+ consumption from the scenario |
| **Project sponsorship** | Customer must provide required environment access and executive sponsorship |

### ✅ Supported Workloads for Nomination

- Power Platform: Copilot Studio
- Power Platform: Copilot Studio (Prepaid Capacity Pack / PayGo / P3)
- Power Platform: AI Builder / Power Virtual Agents
- Azure AI Foundry
- MW: Copilot Agents / Copilot Agents LoB / Copilot Connectors
- Dynamics 365 1P *(nominate with Copilot Studio milestone)*

> 📌 Unified Support is **not required** to access CAF.

## 🤖 Supported Scenarios

CAF supports a broad and growing set of pre-defined, repeatable agent delivery patterns across key business domains:

### Technical Patterns
- LLM output steering and fine-tuning
- AI Search for enhanced accuracy
- Document processing & intelligent extraction
- Knowledge retrieval (B2E)
- Legacy LOB interactions and RPA automation
- Image analysis for service workflows

### Business Scenarios

| Domain | Scenarios |
|---|---|
| **Sales** | RFP response automation, claims processing, lead qualification |
| **Service** | Contact center agents, knowledge lifecycle management, case management |
| **Employee Self-Service (ESS)** | Employee assist bots, time & expense automation, HR onboarding |
| **Finance & Supply Chain** | Supplier communication, PO processing automation |
| **Governance & Security** | Agent Landing Zone setup for enterprise-grade compliance |

> 📌 The scenario catalog continues to expand as more agent deliveries are completed.  
> If a customer scenario falls outside current scope, CAF will route to the most appropriate delivery channel.
